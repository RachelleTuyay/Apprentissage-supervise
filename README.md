# Apprentissage supervise PROJET

Ce projet vise à comparer les performances de plusieurs classifieurs à partir d’un corpus préalablement annoté, avant d’en analyser et d’en interpréter les résultats.

## Corpus : 
Le corpus est une transcription de l'oral d'entretien entre chercheurs et habitants d'Orléans. Il a été annoté à l’aide d’un système binaire d’étiquettes identifiant les questions comme « question canonique » ou « non-question ».
